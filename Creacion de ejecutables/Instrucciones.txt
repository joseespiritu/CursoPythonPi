PRIMERO SE DEBE DE INSTALAR CON CONSOLA EN MODO ADMINISTRADOR

pip install pyinstaller

UNA VEZ INSTALADO
IR A LA CARPETA DONDE ESTA EL MODULO O PROGRAMA
EL PROGRAMA DISTRIBUIBLE QUEDARA EN LA CARPETA /DIST DE LOS ARCHIVOS QUE SE GENEREN
- PARA GENERAR TODOS LOS ARCHIVOS USAMOS EL COMANDO: pyinstaller nombre_archivo_python
- PARA GENERAR EL ARCHIVO SIN LA CONSOLA (SI ES QUE LA LLEGA A TENER): pyinstaller --windowed nombre_archivo_python

SIN EMBARGO NECESITA TODOS LOS ARCHIVOS QUE SE GENERAN
- PARA GENERAR EL ARCHIVO EXE Y QUE NO DEPENDA DE LOS DEMAS ARCHIVOS: pyinstaller --windowed --onefile nombre_archivo_python
- PARA GENERAR EL ARCHIVO EXE Y QUE NO DEPENDA DE LOS DEMAS ARCHIVOS,PERO 
ADEMAS TENGA UN ICONO PERSONALIZADO, EN CASO DE QUE ESTE EN EL MISMO DIRECTORIO: pyinstaller --windowed --onefile --icon=./logo.ico nombre_archivo_python
